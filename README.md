# Tetris in JS

Simple Tetris game made in vanilla Js, starting with Vite template.  
Everything insteresting in this project is in `main.js`.

## Development:

Depends on vite, make sure you have it installed.

`npm install`  
`npm run dev`

## Build production

Creates `/dist` folder for production

`npm run build`

## Preview

`npm run preview`
